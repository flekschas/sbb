Info

The entity panel provides a list of all available biological entities within the current view / illustration. A click on a tag will highlight the corresponding entity in the illustration.

Entities indicated with a small magnitude on the top right are zoom-able. A double-click will lead to a zoom-in.

The list is sorted alphabetically and groups entities that start with the same first character.

A entity is an area of interest within the illustration. This can be an organ, an anatomical structure, a cell or a subcellular component.
