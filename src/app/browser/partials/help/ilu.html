Actions

HighlightClick / Tap

By clicking / tapping on an anatomical entity it will be highlighted as well as its corresponding tag in the list of units on the right.

A drop-down containing information about this entity as well as the possibility to zoom in if available will pop-up.

Infos

A unit is an area of interest within the illustration. This can be for example an organ, an important structure within an organ, a cell or a cellular component.
Units can be identified by hovering with the mouse's cursors. If an area turns yellow it represents a unit.
